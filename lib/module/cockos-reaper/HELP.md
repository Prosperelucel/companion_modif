## Cockos Reaper
This module connects to the Cockos Reaper software.

### Configuration
* Type in the IP address of the computer running Reaper.
* Type in the Port that Reaper is listening for OSC.

### Available actions
* Record
* Play
* Stop
* Pause
* Autoarm Record
* Reset Solos
* Rewind
* Forward
* Click/Metronome
* Go To Marker
* Go To Region
* Track Mute
* Track Solo
* Track Arm
* Track Unmute
* Track Unsolo
* Track Unarm
* Custom Action
