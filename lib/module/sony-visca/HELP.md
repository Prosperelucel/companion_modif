## Sony Visca
This module uses the Sony Visca protocol to control PTZ cameras.

### Configuration
* Type in the IP address of the device.
* Type in the port of the device (default is 52381).
* You can also specify the Camera ID.

* Pan Right
* Tilt Up
* Tilt Down
* Up Left
* Up Right

* Down Right
* P/T Stop

* Zoom In
* Zoom Out
* Zoom Stop

* Focus Far

* One Push Auto Focus
* Set Exposure Mode

* Gain Down

* Iris Down
